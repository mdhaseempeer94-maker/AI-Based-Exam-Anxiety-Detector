# AI Exam Anxiety Detector

The AI Exam Anxiety Detector is an intelligent mental wellness support system that detects exam-related anxiety from student text inputs.

The system uses Natural Language Processing (NLP) and a BERT-based model to analyze student reflections, feedback, and exam thoughts.

It classifies anxiety into three categories:
- Low Anxiety
- Moderate Anxiety
- High Anxiety

Technologies Used:
- Python
- Streamlit
- FastAPI
- Transformers (BERT)
- Scikit-learn
- Pandas
